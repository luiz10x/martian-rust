A trait that represents a MartianFileType that can be read into memory as type T or written from type T. Use the read() and write() methods to achieve these.

If you want to implement this trait for a custom filetype, read the inline comments on which functions are provided and which are required.

Provided methods

fn read(&self) -> Result<T, Error>

Read the MartianFileType as type T The default implementation should work in most cases. It is recommended not to implement this for a custom filetype in general, instead implement read_from()

fn write(&self, item: &T) -> Result<(), Error>

Write type T into the MartianFileType The default implementation should work in most cases. It is recommended not to implement this for a custom filetype in general, instead implement write_into()
